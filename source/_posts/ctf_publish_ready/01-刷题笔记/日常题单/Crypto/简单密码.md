# 分析

猜测flag头是ctfshow{，把它转成16进制

```python
63746673686f777b
```

对比密文可以发现是变种凯撒，写脚本还原。

```python
enc = "647669776d757e83817372816e707479707c888789757c92788d84838b878d9d"

enc = bytes.fromhex(enc)
flag = ''
for i in range(len(enc)):
	tmp = enc[i]-i-1
	flag += chr(tmp)
	
print(flag)
# ctfshow{xiguabei_just_ez_signin}
```
